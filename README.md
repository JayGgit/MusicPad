
## MusicPad

**Made for Hackclub's [Blueprint](blueprint.hackclub.com)**

A super simple music control pad.

Features:
- Skip track
- Rewind track
- Play / Pause
- Mute / Unmute
-  Display - Does nothing for right now. I might change it later to include playback info from the plugged in device.
